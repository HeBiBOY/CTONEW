# 渐渐的个人引导页

一个现代化、响应式的个人引导页（Landing Page）项目，展示了程序员渐渐的技能、项目和联系方式。

## ✨ 特性

- 🎨 **现代化设计** - 使用渐变色和毛玻璃效果
- 📱 **完全响应式** - 在所有设备上完美显示（桌面、平板、手机）
- 🎯 **流畅动画** - 各种进入动画和交互效果
- 🧭 **平滑导航** - 顶部导航栏和平滑滚动
- 📧 **联系表单** - 包含完整的联系表单
- 🌓 **社交媒体链接** - 固定侧边栏社交链接
- ⚡ **快速加载** - 轻量级代码，无依赖库
- 🎭 **交互式元素** - 悬停效果、滚动动画等

## 🚀 快速开始

### 使用本地开发服务器

使用Python 3：
```bash
python -m http.server 8000
```

使用Python 2：
```bash
python -m SimpleHTTPServer 8000
```

使用Node.js（http-server）：
```bash
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`

## 💻 自定义指南

### 修改个人信息

编辑 `index.html`：

```html
<!-- 修改姓名 -->
<h1 class="hero-title">您的名字</h1>

<!-- 修改职位 -->
<p class="hero-subtitle">您的职位</p>

<!-- 修改邮箱 -->
<p>您的邮箱@example.com</p>

<!-- 修改电话 -->
<p>+86 138-1234-5678</p>
```

### 修改颜色主题

编辑 `styles.css` 中的根变量：

```css
:root {
    --primary-color: #667eea;        /* 修改主色 */
    --primary-dark: #764ba2;         /* 修改深色 */
    --secondary-color: #f093fb;      /* 修改辅助色 */
    /* ... 其他变量 */
}
```

### 修改社交媒体链接

编辑 `index.html` 中的社交链接部分，替换 `href="#"` 为实际URL
